# ✍️ DML

Este proyecto contiene comandos DML (Data Manipulation Language) en SQL.

Los comandos DML permiten **manipular los datos** existentes en la base de datos, es decir, agregar, actualizar o eliminar registros.

## 📂 Proyectos Personales

* 👉 [Comando INSERT](./1.%20INSERT)
* 👉 [Comando UPDATE](./2.%20UPDATE)
* 👉 [Comando DELETE](./3.%20DELETE)

---

📌 **Objetivo:** Practicar la inserción, modificación y eliminación de registros para mantener la integridad de los datos.

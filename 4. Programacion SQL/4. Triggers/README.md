# ⚡ Triggers  

Los *Triggers* en SQL son **procedimientos automáticos** que se ejecutan en respuesta a eventos en una tabla, como `INSERT`, `UPDATE` o `DELETE`.  

Permiten validar datos, mantener integridad y automatizar acciones.  

📌 Se ejecutan automáticamente sin necesidad de llamarlos manualmente.

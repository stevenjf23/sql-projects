# ⚙️ Stored Procedures (Procedimientos Almacenados)  

Los *Stored Procedures* en SQL son **bloques de código almacenados** en la base de datos que pueden ejecutarse cuando se necesiten.  

Permiten automatizar tareas, reutilizar lógica y mejorar el rendimiento.  

📌 Son útiles para centralizar procesos y operaciones complejas.

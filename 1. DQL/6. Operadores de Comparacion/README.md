# ⚖️ Operadores de Comparación  

Los operadores de comparación en SQL se utilizan para **evaluar condiciones** dentro de una consulta, principalmente en la cláusula `WHERE`.  

Permiten comparar valores y filtrar resultados según criterios específicos.  

## Tipos:  
- `=` Igualdad  
- `<>` Desigualdad  
- `<` Menor que  
- `>` Mayor que  
- `<=` Menor o igual que  
- `>=` Mayor o igual que  

📌 Son fundamentales para realizar consultas precisas.

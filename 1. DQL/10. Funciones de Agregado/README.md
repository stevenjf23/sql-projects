# 📊 Funciones de Agregado  

Las funciones de agregado en SQL se utilizan para **realizar cálculos sobre un conjunto de datos** y devolver un único resultado.  

Son comúnmente utilizadas junto con `GROUP BY` para resumir información.  

## Tipos:  
- `SUM()` → Suma de valores  
- `COUNT()` → Cantidad de registros  
- `AVG()` → Promedio  
- `MIN()` → Valor mínimo  
- `MAX()` → Valor máximo  

📌 Son esenciales para el análisis y resumen de datos.

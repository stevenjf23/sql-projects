# 🏅 Función DENSE_RANK  

La función `DENSE_RANK()` en SQL se utiliza para **asignar una posición** a cada fila dentro de un conjunto de resultados.  

Se usa junto con `OVER()` y `ORDER BY`, y asigna el mismo rango a valores iguales **sin dejar espacios** en la numeración.  

📌 Es útil para clasificaciones continuas sin saltos.

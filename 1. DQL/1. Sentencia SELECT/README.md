# 🔍 Sentencia SELECT  

La sentencia `SELECT` es utilizada en SQL para **consultar y obtener datos** de una base de datos.  

Permite seleccionar columnas específicas de una o varias tablas, así como aplicar filtros, ordenamientos y agrupaciones.  

📌 Es la base del lenguaje DQL y una de las consultas más utilizadas en SQL.

# 🎯 Cláusula WHERE  

La cláusula `WHERE` se utiliza en SQL para **filtrar los registros** que cumplen una condición específica.  

Permite seleccionar únicamente los datos que coinciden con ciertos criterios dentro de una consulta.  

📌 Es fundamental para trabajar con datos de forma precisa.

# 🔀 Cláusulas UNION y UNION ALL  

Las cláusulas `UNION` y `UNION ALL` se utilizan en SQL para **combinar los resultados de dos o más consultas** en un solo conjunto de resultados.  

## Tipos:  
- `UNION` → Combina resultados eliminando los duplicados  
- `UNION ALL` → Combina resultados incluyendo duplicados  

📌 Las consultas deben tener el mismo número de columnas y tipos de datos compatibles.

# 🔀 Función IF  

La función `IF` en SQL se utiliza para **evaluar una condición** y devolver un valor si se cumple, u otro si no se cumple.  

Permite aplicar lógica condicional directamente en las consultas.  

📌 Es útil para personalizar resultados según ciertas condiciones.

# 🔢 Funciones Matemáticas  

Las funciones matemáticas en SQL se utilizan para **realizar cálculos numéricos** dentro de una consulta.  

Permiten ajustar o redondear valores según sea necesario.  

## Tipos:  
- `CEILING()` → Redondea hacia arriba al entero más cercano  
- `FLOOR()` → Redondea hacia abajo al entero más cercano  
- `ROUND()` → Redondea un número al número de decimales especificado  

📌 Son útiles para trabajar con datos numéricos y financieros.

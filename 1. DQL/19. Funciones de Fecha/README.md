# 📅 Funciones de Fecha  

Las funciones de fecha en SQL se utilizan para **trabajar y manipular datos de tipo fecha** dentro de una base de datos.  

Permiten extraer partes de una fecha o realizar cálculos con ellas.  

## Tipos:  
- `DAY()` → Obtiene el día  
- `MONTH()` → Obtiene el mes  
- `YEAR()` → Obtiene el año  
- `DATE_ADD()` → Suma un intervalo de tiempo a una fecha  
- `DATEDIFF()` → Calcula la diferencia entre dos fechas  

📌 Son esenciales para el manejo y análisis de información temporal.

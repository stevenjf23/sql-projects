# 🔢 Función ROW_NUMBER  

La función `ROW_NUMBER()` en SQL se utiliza para **asignar un número consecutivo** a cada fila de un resultado.  

Se usa junto con `OVER()` para definir el orden y partición de los datos.  

📌 Es útil para numerar registros, paginación y análisis de datos.

# 🔀 Función CASE  

La función `CASE` en SQL se utiliza para **evaluar múltiples condiciones** y devolver un resultado según se cumplan.  

Permite aplicar lógica condicional más compleja dentro de una consulta.  

📌 Es una alternativa más flexible que `IF` para manejar varios escenarios.

# 🔗 Cláusulas JOIN (INNER, LEFT, RIGHT)  

Las cláusulas `JOIN` en SQL se utilizan para **combinar datos de múltiples tablas** basándose en una relación entre ellas.  

## Tipos:  
- `INNER JOIN` → Devuelve solo los registros que tienen coincidencia en ambas tablas  
- `LEFT JOIN` → Devuelve todos los registros de la tabla izquierda y los coincidentes de la derecha  
- `RIGHT JOIN` → Devuelve todos los registros de la tabla derecha y los coincidentes de la izquierda  

📌 Son esenciales para trabajar con bases de datos relacionales.

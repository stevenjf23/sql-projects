# 🔄 Función CAST  

La función `CAST` en SQL se utiliza para **convertir un dato de un tipo a otro**.  

Permite cambiar el formato de los datos para que sean compatibles en una consulta.  

📌 Es útil para manejar y transformar datos dentro de operaciones SQL.

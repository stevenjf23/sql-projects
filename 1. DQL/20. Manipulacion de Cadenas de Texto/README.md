# 🔤 Funciones de Manipulación de Cadenas  

Las funciones de texto en SQL se utilizan para **trabajar y transformar cadenas de caracteres** dentro de una consulta.  

Permiten modificar, combinar o extraer partes de un texto.  

## Tipos:  
- `UPPER()` → Convierte a mayúsculas  
- `LOWER()` → Convierte a minúsculas  
- `CONCAT()` → Une cadenas de texto  
- `SUBSTRING()` → Extrae una parte del texto  
- `LENGTH()` → Obtiene la longitud del texto  
- `LEFT()` → Extrae caracteres desde la izquierda  
- `RIGHT()` → Extrae caracteres desde la derecha  

📌 Son útiles para limpiar y formatear datos de texto.

# 📋 Cláusula IN y NOT IN  

Las cláusulas `IN` y `NOT IN` se utilizan en SQL para **filtrar registros** comparando un valor con una lista de valores.  

- `IN` → Selecciona los registros que coinciden con alguno de los valores de la lista  
- `NOT IN` → Selecciona los registros que **no** coinciden con los valores de la lista  

📌 Son útiles para simplificar múltiples condiciones en una consulta.

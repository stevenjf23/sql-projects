# 🧩 CTE (Common Table Expressions)  

Las CTEs en SQL son consultas temporales que se definen usando `WITH` y se utilizan dentro de una consulta principal.  

Permiten **organizar y simplificar consultas complejas**, haciéndolas más legibles y reutilizables.  

📌 Son útiles para dividir consultas grandes en partes más claras.

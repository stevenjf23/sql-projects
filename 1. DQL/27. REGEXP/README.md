# 🔍 Operador REGEXP  

El operador `REGEXP` en SQL se utiliza para **buscar patrones avanzados** en cadenas de texto mediante expresiones regulares.  

Permite realizar coincidencias más complejas que `LIKE`, como validaciones de formato o búsquedas específicas.  

📌 Es útil para filtrar datos con patrones personalizados.

# 🔎 Cláusula LIKE  

La cláusula `LIKE` se utiliza en SQL para **buscar patrones de texto** dentro de una columna.  

Permite realizar búsquedas parciales usando comodines como `%` (cualquier conjunto de caracteres) y `_` (un solo carácter).  

📌 Es útil para encontrar coincidencias aproximadas en los datos.

# 🏆 Función RANK  

La función `RANK()` en SQL se utiliza para **asignar una posición** a cada fila dentro de un conjunto de resultados.  

Se usa junto con `OVER()` y `ORDER BY`, y asigna el mismo rango a valores iguales, dejando saltos en la numeración.  

📌 Es útil para clasificaciones y comparaciones dentro de grupos de datos.

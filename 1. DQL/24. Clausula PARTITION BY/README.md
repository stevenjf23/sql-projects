# 🧩 Cláusula PARTITION BY  

La cláusula `PARTITION BY` se utiliza en SQL para **dividir los datos en grupos** dentro de funciones de ventana.  

Permite aplicar cálculos de forma independiente en cada grupo.  

📌 Es útil para analizar datos sin perder el detalle de cada fila.

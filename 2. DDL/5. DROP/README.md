# 🗑️ Sentencia DROP  

La sentencia `DROP` en SQL se utiliza para **eliminar objetos de la base de datos**, como tablas, bases de datos o vistas.  

Al ejecutarla, se borra completamente la estructura y los datos del objeto.  

📌 Debe usarse con precaución, ya que no se puede deshacer fácilmente.

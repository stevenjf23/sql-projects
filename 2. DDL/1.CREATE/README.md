# 🏗️ Sentencia CREATE 

La sentencia `CREATE` en SQL se utiliza para **crear objetos en la base de datos**, como tablas, bases de datos, vistas o índices.  

Forma parte del lenguaje DDL (Data Definition Language).  

📌 Es fundamental para definir la estructura de una base de datos.

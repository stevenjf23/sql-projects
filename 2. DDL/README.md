# 🏗️ DDL

Este proyecto contiene comandos DDL (Data Definition Language) en SQL.

Los comandos DDL permiten **definir, modificar y gestionar la estructura** de los objetos en una base de datos.

* 👉 [Comando USE](./2.%20USE)
* 👉 [Comando DESCRIBE](./3.%20DESCRIBE)
* 👉 [Comando ALTER](./4.%20ALTER)
* 👉 [Comando DROP](./5.%20DROP)

📌 **Objetivo:** Practicar la creación y administración de estructuras de datos de manera organizada.

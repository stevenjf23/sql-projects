# 📋 Sentencia DESCRIBE  

La sentencia `DESCRIBE` en SQL se utiliza para **mostrar la estructura de una tabla**.  

Permite visualizar información como nombres de columnas, tipos de datos y restricciones.  

📌 Es útil para conocer la definición de una tabla antes de consultarla o modificarla.

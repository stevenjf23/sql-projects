# 🗂️ Sentencia USE  

La sentencia `USE` en SQL se utiliza para **seleccionar la base de datos activa** sobre la cual se ejecutarán las consultas.  

Permite indicar en qué base de datos se desea trabajar antes de realizar operaciones.  

📌 Es fundamental para dirigir correctamente las consultas.
